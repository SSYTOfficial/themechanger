# phpBB Theme Changer Module
PhpBB Theme Changer Extension

# Install
`Check on folder ext exist folder 'ssyt'`
```
ext/ssyt -> themechanger
```

# Activation
`ACP -> Customise -> Extensions Manager -> Theme Changer Module -> Enable`
